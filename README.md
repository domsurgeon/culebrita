# CulebrAI

Snake game solved with Deep Q-Learning using TensorFlow.js.

### Requires

- [Node v18.13.0](https://nodejs.org/)
- npm v9.3.1

### Installation

```sh
$ npm ci
```

### Usage

```sh
$ npm run start # serves game at http://localhost:3000 and starts agent at http://localhost:5000 concurrently
```
